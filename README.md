# RepoBeacon

A VS Code extension for prominently displaying repository names with animated visual styles. "Peacock for people who need more."

Originally prototyped in Figma: https://www.figma.com/design/E6trm1Og0mDWqhLuCIhlCX/Distinctive-Repo-Name-Displays

The current codebase is a React/Vite demo showcasing the 15 animation styles.

```bash
npm run build  # Production build
```
